# Public Higher Ed Institutions

A structured dataset of public higher education institutions.

This repository is intended for use in data projects that require reliable, organized lists of higher education institutes for research, analytics, applications, and institutional analysis.

---

## Overview

This repository provides:

- Cleaned and standardized institution names
- Database-friendly formats
- ASCII-normalized versions where needed
- Structured data suitable for ingestion into applications or analytics pipelines

The goal is to make higher education institutional data easy to use, integrate, and scale.

## Use Cases

This dataset is useful for:

- Higher education research
- EdTech applications
- Institutional analytics
- CRM enrichment
- Market research
- Geographic mapping projects
- Data normalization pipelines

## Data Standards

- UTF-8 encoded
- Deduplicated entries
- Cleaned formatting
- Optional ASCII-normalized version for strict database systems
